## O que é RádioBS?

RádioBS é um sistema incrível para web rádio com layout responsivo com todos os recursos de que você precisa para iniciar sua rádio na web hoje.
O sistema RádioBS foi lançado em 2016 e até hoje é mantido com atualizações constantes, utilizando um poderoso sistema de atualização automática.
RádioBS conta com os principais recursos que uma rádio precisa, por isso, o sistema torna-se o favorito para grandes empresas de hospedagem.
